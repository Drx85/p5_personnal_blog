Project started on 03/27/2021 - ended on 04/04/2021
Used langages : HTML, PHP, SQL

Blog features :
- MVC architecture
- Display posts and linked comments from SQL data-bases
- Display old posts on different pages (automatic pagination)
- Display number of comments for each post + inform if there is no comment on a post
- Date and time of sent post or comment
- Add comment by user, notify and doesn't add comment if pseudo or text comment are empty
- Code secure using htmlspecialchars and isset/empty/interval condition on POST and GET variables
- Administration environment to add/edit/delete posts, and to delete comments
- Administration environment protected by .htaccess / .htpasswd (encrypted password is recommended but doesn't work on WAMP)

Used tools :
- PhpStorm
- WAMP
- Git
